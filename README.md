# Projet banque réalisé dans le cadre de l'UE NFA035 Java bibliothèque et pattern avec le CNAM

## Auteur : *Mickaël L*

## Avancement du projet 
Le projet est fonctionnel. 
Un bug a corriger : lorsque le choix 2 est sélectionné et que l'on sélectionne ensuite opération bancaire dans le menu, la fenêtre est doublée.

## Description et objectif  
Réaliser une application simulant les opérations de gestion de compte bancaire. Réalisation d'une IHM, architecturer le projet (MVC), utilisation massive de la documentation Oracle en implémentant les méthodes existantes.

## Pour lancer l'application
Java Runtime Environment : JRE 9.0.1
* java -jar banque.jar

## Disponible dans ce projet 
* un executable => banque.jar
* un diagramme de classe => ClassDiagram.jpg
* une documentation => javaDoc
