######################### Banque #########################

Projet réalisé dans le cadre de l'untité d'enseignement NFA035 Java bibliothèque et pattern avec le CNAM

Objectif : Réaliser une application simulant les opérations de gestion de compte bancaire. Réalisation d'une IHM, architecturer le projet (MVC), utilisation massive de la documentation Oracle en implémentant les méthodes existantes.

Java Runtime Environment : JRE 9.0.1

Disponible dans ce projet :
- un executable => Banque.jar
- un diagramme de classe => assets/ClassDiagram.jpg
- une documentation => javaDoc (dossier)
- Les sources
